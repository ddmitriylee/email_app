# Lab 2 Email App

## Installation

- Install dependencies

```
npm install
```

- Create <strong>.env</strong> file

- Configure <strong>.env</strong> file 

```
PORT=your.port
EMAIL=your.email@gmail.com
APPPASS=your gmail passcode
```

- Initialize the application

```
npm run dev
```

## Dependencies Used

- dotenv ^16.4.5
- ejs ^3.1.9
- express ^4.18.2
- nodemailer ^6.9.9
- nodemon ^3.0.3
